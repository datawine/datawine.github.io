http_load是一个非常轻量级的工具，基于linux平台测试网站的性能，体积非常小，通过并行复用的方式运行，可以测试web服务器的吞吐量与负载。

安装与使用

安装

非常简单。

然后编辑

使用

编辑一个文件，存放需要访问的网页，如http://www.baidu.com，文件名存为url

运行命令

参数

  • -p
    • 并发的用户进程数
  • -f
    • 总计的访问次数
  • -r
    • 每秒的访问频率
  • -s
    • 连续的访问时间
  • url
    • 存放网站连接或者url的文件
